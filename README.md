# easyCreative
一，功能：实现通过一个有效的iTunes包名获取该应用的素材。

二，用法：
    
    1，require './vendor/autoload.php'; 若用框架，则忽略这一步。
    
    2，use EasyCreative\Kernel; use 内核文件，与框架冲突请 as 别名。
    
    3，Kernel::StartCrawling('297606951'); 传入一个有效的包名。
